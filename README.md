# University Management System 🎓
نظام إداري للجامعات مبني بـ **ASP.NET Core MVC** لإدارة الطلاب والكورسات.

## ✨ الميزات (Features)
- **CRUD Operations:** إدارة كاملة للطلاب، الكورسات، والأقسام.
- **UI/UX:** واجهة مستخدم متفاعلة باستخدام **Razor Views**.
- **Data Validation:** التأكد من صحة البيانات المدخلة من جهة السيرفر.
- **Relational DB:** التعامل مع علاقات قواعد البيانات المعقدة (Many-to-Many).

## 🛠 التقنيات (Tech Stack)
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML/CSS & Bootstrap
